Guided Exercise: HTML, CSS, and JavaScript Unit Converter

In this exercise, we will introduce HTML, CSS, and JavaScript by creating a simple unit converter. The project is divided into manageable steps, allowing for incremental learning. Each step involves adding a piece of code and explaining its functionality.

Steps

Step 1: Basic HTML Structure

Code:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Unit Converter</title>
</head>
<body>
</body>
</html>

Explanation:

Start by creating the basic HTML structure. The <!DOCTYPE html> declares the document as HTML5. The <head> contains metadata like the page title, while the <body> will contain visible content.

Step 2: Add the Converter Container

Code:

<div id="container">
    <h1>Unit Converter</h1>
</div>

Explanation:

Add a container for the converter using the <div> tag. Inside the container, include a heading (<h1>) to indicate the page's purpose.

Step 3: Add Input Fields

Code:

<label for="dimension">Type:</label>
<select name="dimension" id="dimension">
    <option>length</option>
    <option>mass</option>
</select>

<label for="value">Value:</label>
<input id="value" type="number">

<label for="from">From:</label>
<select name="from" id="from"></select>

<label for="to">To:</label>
<select name="to" id="to"></select>

Explanation:

Add dropdowns (<select>) for selecting the type (length or mass) and the units (from and to). Include an input field (<input>) to accept a numeric value for conversion.

Step 4: Add Result Display

Code:

<h3>Result:</h3>
<h4 id="result"></h4>

Explanation:

Add headings for displaying the conversion result. Use an empty <h4> with an id (result) to dynamically update and show the output.

Style the container with borders, rounded corners, and a shadow. Use flex to align elements and add spacing between them.

Step 6: Add JavaScript for Dynamic Dropdowns

Code:

<script>
let quantity = document.getElementById("dimension");
let from = document.getElementById("from");
let to = document.getElementById("to");

function updateSelection() {
    from.innerHTML = '';
    to.innerHTML = '';

    let options;
    if (quantity.value === "length") {
        options = ["meter", "kilometer"];
    } else if (quantity.value === "mass") {
        options = ["gram", "kilogram"];
    }

    options.forEach(unit => {
        let fromOption = document.createElement("option");
        let toOption = document.createElement("option");
        fromOption.textContent = unit;
        toOption.textContent = unit;
        from.appendChild(fromOption);
        to.appendChild(toOption);
    });
}

quantity.addEventListener("change", updateSelection);
updateSelection();
</script>

Explanation:

Dynamically populate the from and to dropdowns based on the selected type (length or mass). Use an event listener to update the dropdowns whenever the type changes.

Step 7: Add Conversion Logic

Code:

<script>
function convert() {
    let value = parseFloat(document.getElementById("value").value);
    let fromUnit = document.getElementById("from").value;
    let toUnit = document.getElementById("to").value;
    let resultElement = document.getElementById("result");
    let result;

    if (document.getElementById("dimension").value === "length") {
        if (fromUnit === "meter" && toUnit === "kilometer") {
            result = value / 1000;
        } else if (fromUnit === "kilometer" && toUnit === "meter") {
            result = value * 1000;
        } else {
            result = value; // Same unit
        }
    } else if (document.getElementById("dimension").value === "mass") {
        if (fromUnit === "gram" && toUnit === "kilogram") {
            result = value / 1000;
        } else if (fromUnit === "kilogram" && toUnit === "gram") {
            result = value * 1000;
        } else {
            result = value; // Same unit
        }
    }

    resultElement.textContent = result ? result.toFixed(5) : "Invalid input";
}

document.getElementById("value").addEventListener("input", convert);
document.getElementById("from").addEventListener("change", convert);
document.getElementById("to").addEventListener("change", convert);
</script>

Explanation:

This function performs the unit conversion based on the selected type (length or mass) and units (from and to). The event listeners ensure that the conversion is automatically triggered whenever the input value or selected units are changed.

Breakdown:

  • Retrieve the input value and selected units from the DOM.
  • Perform conversions based on the selected dimension (length or mass).
  • Update the result element dynamically with the computed value.
